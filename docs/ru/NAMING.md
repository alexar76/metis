# Именование: Metis

## Почему Metis

**Metis** (греч. μῆτις, *mētis*) — богиня совета, мудрости и глубокой мысли. Название отражает архитектуру проекта:

- **Understanding Council** — параллельная интерпретация задачи несколькими советниками
- **Распределённое познание** — рассуждение на узлах и моделях
- **Мудрость важнее скорости** — маршрутизация, верификация и пороги уверенности

## Место в экосистеме

Экосистема [alexar76](https://github.com/alexar76) использует мифологические имена:

| Проект | Мифология | Роль |
|--------|-----------|------|
| **Helios** | Бог солнца | Наблюдаемость |
| **Argus** | Всевидящий великан | Агент спроса / клиент |
| **Dioscuri** | Божественные братья | Парные сервисы |
| **Metis** | Богиня совета | Слой рассуждения и оркестрации |

Metis находится **над** сырыми LLM-эндпоинтами и **под** агентами вроде Argus.

## Имена моделей API

Идентификаторы OpenAI-совместимых моделей: `metis-<маршрут>`:

| Модель | Маршрут | Назначение |
|--------|---------|------------|
| `metis` | Авто | Классификатор выбирает глубину |
| `metis-fast` | Fast | Минимальная задержка |
| `metis-thinking` | Thinking | Расширенное рассуждение |
| `metis-council` | Council | Совет + MoA |
| `metis-agent` | Agent | Агентный цикл с инструментами |

## Переменные окружения

Префикс `METIS_` (например `METIS_API_KEY`). На один релиз также принимаются `SUPERBRAIN_*` и `COGNITIVE_*`.

## RPC распределённых узлов

- `GET /metis/health`
- `POST /metis/invoke`

Заголовки подписи: `X-Metis-Timestamp`, `X-Metis-Signature`.

## Python API

```python
from metis import Metis, RuntimeConfig
```

Алиасы: `Superbrain`, `CognitiveExoskeleton`.
